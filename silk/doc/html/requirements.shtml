what software does silk require?

silk was primarily developed and tested using the following setup:
  • Perl version 5.005_03
  • CGI.pm version 2.4.6
  • MD5.pm version 1.7

silk has also been tested using following setup (the the silk Sandbox):
  • Perl version 5.8.0
  • CGI.pm version 2.8.9
  • MD5.pm version 2.20
